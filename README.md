# sillytavern-Hou-chuli
一个用于 SillyTavern 的插件。
插件简介
本插件为 SillyTavern 底部栏扩展工具，支持从当前聊天最后一条消息中按标签截取文本、套用提示词模板、手动发送给 AI，并在确认后将 AI 回复替换回当前聊天内容。适用于文本提取、润色、总结、二次加工等场景。

使用步骤
点击顶部栏插件按钮打开面板。
在“设置范围”中配置开始标签、结束标签，可选是否保留标签。
在“提示词模板”中选择或编辑需要套用的模板。
在“截取与发送”中点击“手动触发”，将最后一条消息内容提取到右侧文本框。
检查或修改右侧文本框内容后，点击“手动发送”。
AI 返回结果后，在确认框中检查内容并点击确认，即可替换当前聊天最后一条消息。
